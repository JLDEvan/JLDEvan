## Projeto em destaque

**Sistema de monitoramento de inversores fotovoltaicos (privado)**  

Sistema de monitoramento em tempo real de inversores fotovoltaicos com detecção de anomalias utilizando machine learning.

Atuei como único desenvolvedor, sendo responsável por:
- Desenvolvimento de APIs com FastAPI
- Integração com modelos de ML (scikit-learn)
- Processamento e análise de dados operacionais
- Modelagem de banco de dados (PostgreSQL)
- Containerização com Docker
